# Simple Blog Application

This is a simple blog application built with Python and Flask. It includes placeholder files for posts, routing, and templates, organized in a clean folder structure.

## Project Structure

- `app.py`: Main Flask application file.
- `requirements.txt`: List of Python dependencies.
- `data_structures.py`: Implementations of Stack and Queue data structures.
- `blog/`: Blog-related code.
  - `models/`: Data models.
    - `post.py`: Post model class.
  - `views/`: View functions.
    - `blog_views.py`: Blog view functions.
  - `templates/`: HTML templates.
    - `base.html`: Base template.
    - `index.html`: Home page template.
    - `post.html`: Individual post template.

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python app.py`
3. Open your browser to `http://127.0.0.1:5000/`

## Data Structures

The `data_structures.py` file contains implementations of Stack and Queue classes, as per the coding challenges.

### Stack
- `is_empty()`: Check if stack is empty.
- `push(item)`: Add item to stack.
- `pop()`: Remove and return top item.
- `peek()`: Return top item without removing.
- `size()`: Get number of items.

### Queue
- `is_empty()`: Check if queue is empty.
- `enqueue(item)`: Add item to queue.
- `dequeue()`: Remove and return front item.
- `size()`: Get number of items.
